# CSV Data Cleaning Pipeline

## Overview

This project demonstrates basic data cleaning and preprocessing using Python and pandas. The goal is to convert raw, messy CSV data into a clean and usable format.

## Features

* Read CSV file using pandas
* Handle missing values
* Validate data (age range and email format)
* Remove or store invalid data
* Standardize text data (lowercase emails, formatted names)
* Create new column (Age Group)
* Save cleaned dataset to a new CSV file

## Technologies Used

* Python
* Pandas
* Regular Expressions (re)

## Input

A raw CSV file containing:

* Name
* Email
* Age
* City

## Output

* Cleaned CSV file (`cleaned_data.csv`)
* Invalid data file (`invalid_data.csv`)

## Workflow

1. Load data from CSV
2. Handle missing values
3. Validate data (age and email)
4. Remove invalid records
5. Transform and standardize data
6. Create derived column (Age Group)
7. Save cleaned data
